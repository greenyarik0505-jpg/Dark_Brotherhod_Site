# ⚔️ Brawl Stars Club — Священная Империя 🏰

Официальный веб-сайт Brawl Stars клуба **«Священная Империя»**. 
Красивый, современный одностраничный портал с адаптивным неоновым дизайном, динамической галереей, интерактивным таймлайном событий и функциональной панелью администратора (CRUD).

---

## ✨ Особенности проекта
*   🌌 **Неоновый киберпанк дизайн:** Эффекты Glassmorphism, интерактивное свечение, адаптивные анимации.
*   📜 **Раздел правил:** Удобный просмотр правил клуба и должностных ролей.
*   👥 **Список участников:** Динамическая таблица игроков с сортировкой и фильтрацией.
*   📅 **События:** Интерактивный таймлайн будущих клубных ивентов.
*   🛡️ **Панель управления (Админка):** 
    *   Создание, редактирование и удаление новостей, участников и событий.
    *   **Галерея с загрузкой файлов:** Прямая загрузка изображений с ПК (через File Input или Drag-and-Drop) с конвертацией в Base64.
*   🔒 **Безопасность:**
    *   Все административные CRUD-функции защищены проверкой авторизации.
    *   Пароль скрыт и подключается динамически через секреты GitHub.

## 🚀 Настройка и деплой на GitHub

Этот проект настроен для автоматической сборки и публикации через **GitHub Actions** на **GitHub Pages** с поддержкой скрытого хранения паролей.

### 1. Создание секретов репозитория на GitHub
Чтобы настроить ваш собственный пароль админки, выполните следующие шаги:
1. Зайдите в ваш созданный репозиторий на GitHub.
2. Перейдите во вкладку **Settings** (Настройки) -> **Secrets and variables** -> **Actions**.
3. Нажмите кнопку **New repository secret** (Новый секрет репозитория).
4. Укажите следующие параметры:
    *   **Name (Имя):** `ADMIN_PASSWORD`
    *   **Value (Значение):** Ваш секретный пароль в открытом виде (например, `empire2026`).
5. Нажмите **Add secret**.

### 2. Загрузка проекта в GitHub
Выполните команды в терминале из папки проекта:
```bash
git add .
git commit -m "Initial commit with workflow and README"
git branch -M main
git remote add origin https://github.com/greenyarik0505-jpg/Brawl_Stars_Club_Site.git
git push -u origin main
```
После этого GitHub Actions автоматически запустит сборку, сгенерирует файл конфигурации с вашим секретным паролем из настроек и опубликует сайт на ветку `gh-pages` для GitHub Pages.
